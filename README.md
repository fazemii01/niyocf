
# Niyocf

A project for digitalize cafe shop


## Installation


```bash
  git clone https://github.com/fazemii01/niyocf
  cd app
  npm Install
  npm run build
  npm run start
  cd serve 
  npm install
  npm run app
```
    
## Tech Stack

**Client:** React, Redux, TailwindCSS, RouterDOM, React Hot Toast

**Server:** Node, Express
